# ljg-map：产业地图卡

输入一个行业，输出一张可导航的地图——价值怎么流动、卡在哪、钱沉在哪、未来在哪决战。

## 思想地基：参考系理论（《千脑智能》）

Hawkins 的底：大脑认识任何东西，都是把它的特征钉在一个**参考系**（一张地图）的位置上，靠移动和预测来学。看一个行业也一样。没有地图时，一个行业在脑子里是一团模糊的公司名单；一旦摊成地图——每个环节钉在它的坐标上——原先看不见的结构就一眼可见：价值往哪流、在哪收窄、利润在哪沉淀。

**这张地图不是流程图。** 流程图只画「谁连着谁」；参考系地图画的是**位置关系**——谁在上游谁在下游、谁卡着谁的脖子、谁离钱近谁离钱远。位置说得出几何理由，才算地图。画不出位置关系，就是退化成了装饰方框。

> 一句话立框（thesis，必出现在卡上）：**把一个行业摊成地图，你才看得见——流量在哪收窄（瓶颈），利润在哪沉淀（价值捕获）。而这两处，常常不是同一处。**

## 灵魂：两类节点 + 它们的错位

整张卡的命，是让两类节点一眼可辨，**并让它们的错位现形**。

### 瓶颈节点（流量在此收窄）

行业里被卡住的环节——产能、流量、控制权在这里收窄。识别信号：
- **稀缺产能**：扩产要数年、要巨额资本（晶圆厂、锂矿）。
- **单点控制**：一两家把着，绕不过去（ASML 光刻机、Google 搜索入口）。
- **牌照/专利/标准闸**：法律或技术标准设的关卡（医药审批、CUDA 生态）。
- **物理/认知极限**：带宽、延迟、注意力、信任——绕不开的天花板。

一句话测试：**「这个行业要扩大产出，最先卡住的是哪一环？」** 那一环就是瓶颈。

### 价值捕获节点（利润在此沉淀）

钱实际留下来的环节。注意：**创造价值 ≠ 捕获价值**。识别信号：
- **议价权**：上下游都得求它，它说了算（操作系统、平台收租）。
- **高毛利且持续**：利润率高、还守得住（品牌、专利药、网络效应）。
- **赢家通吃**：规模/网络效应让头部吃掉大部分利润。

一句话测试：**「这个行业赚到的钱，最后沉在哪一环的口袋里？」** 那一环就是价值捕获。

### 错位 —— 卡的全部价值

瓶颈和价值捕获**是两个独立的量**，落在地图上经常**不在同一个节点**：
- **重合**：卡脖子的就是赚钱的（ASML、台积电）——权力与利润同源。
- **错位**：创造价值的环节被卡着、利润被另一环抽走（内容创作者 vs 平台；硬件厂商 vs 英伟达）。

**错位本身就是这个行业的权力地图。** 当 🔴 瓶颈 与 🟡 价值捕获 落在不同节点，在地图上用一条虚线把它们连起来，点破「价值在这创造、钱在那捕获」。这是参考系理论最锋利的兑现：权力结构这个看不见的东西，变成图上一段看得见的距离。

## 三个大问题

地图画完，提炼正好**三个**决定这个行业未来的「大问题」。不是 FAQ，不是科普，是**未决的张力**——答案还没定、一旦定了就改写格局的问题。它们往往落在地图的前沿：瓶颈会不会被技术绕过？价值池会不会迁移？哪个新玩家在改写流向？

每个大问题一句话，锋利、可争论、指向未来。三个之间不重叠，合起来覆盖这个行业最关键的不确定性。判定标准与提炼法见 `references/research.md`。

## 基准率（base rate）：地图的刻度

地图给你结构（谁在哪），但一张没有刻度的地图会被个案带偏——看见一个独角兽就以为遍地黄金。**base rate 是这张地图的刻度尺**：这个行业最关键三个指标的基线值，是你在没有任何额外信息时该持有的先验。继刚的贝叶斯 stance——先验先于个案，base rate 就是被个案锚定的印象的解毒剂。

参考系三件套各管一段，合起来才完整：**地图 = 结构（在哪 / where）**、**base rate = 刻度（多少、多大概率 / how much）**、**三个大问题 = 前沿（接下来 / next）**。卡上顺序也照这条走：先看结构，再用真实刻度校准，最后才往前问——大问题站在校准过的地面上，不悬空。

选哪三个指标，判据三连（都要过）：
- **决策相关**：知道这个数，会改变你怎么下注。
- **反直觉**：大众脑中的印象偏离它（被个案/幸存者偏差带歪）。
- **可查证**：真有数字，能 deep research 查到（区间也行），不编。

每个 base rate 三字段：**指标名 + 基准数值（大字，可为区间/百分比）+ 一句「这意味着」**——点破它戳掉的那个幻觉（如「千里挑一，别拿例外当常态」）。缺「这意味着」就退化成枯燥数字墙；缺数值就是空话。识别与查证法见 `references/research.md`。

## 流程

```
输入：行业/领域名（可附用户已有的判断）
  ↓
1. deep research：价值链结构、关键环节、利润分布、卡点、未决问题，外加**三个关键指标的 base rate**（见 references/research.md —— 研究什么、怎么扇出、怎么复核）
2. 选地图拓扑：价值流（默认）/ 分层技术栈 / 生态网络（按行业结构定，见 research.md）
3. 钉节点：把关键环节钉到地图位置上（节点=功能位，不是公司；公司作占位者标注）
4. 标两类节点：🔴 瓶颈 + 🟡 价值捕获；若错位，画虚线连线点破
5. 判断头像入图：**看图人在这张地图上有没有明确位置？** 有（继刚/用户是这行玩家，或问『我在哪』）→ 在那个节点嵌「你在这里」墨像；无（陌生行业鸟瞰）→ 不放（见 visual.md 判据与画法）
6. 提炼三个 base rate（指标 + 数值 + 「这意味着」）
7. 提炼三个大问题
8. 选 accent 色（按行业温度，见 references/visual.md 调色板）
9. 画地图 SVG {{MAP_SVG}}（复用 visual.md 的 defs 组件 + 骨架；含可选「你在这」墨像）
10. 填 assets/map_template.html 占位变量（含 {{BASE_RATES}}）—— **{{ACCENT}} 最后替换**（SVG 内也含该 token）
11. 渲染（capture.js，fullpage 自适应高度）
12. Read 自验（地图板：节点 placed、双 markers、错位连线；base rate 三指标数值醒目；头像若有则清晰指向节点；三大问题在卡上）→ 交付路径
```

## 视觉规格

## 研究规格

执行前**先 Read `references/research.md`**——研究一个行业要问的问题清单、三种拓扑怎么选、瓶颈与价值捕获节点的识别信号、三大问题的提炼标准、研究 agent 扇出纪律（同步返回、批量错峰、ID 抽样复核）。

## 模板变量（map_template.html）

| 变量 | 内容 |
|------|------|
| `{{ACCENT}}` | 行业温度色 hex（按 visual.md 调色板选，如 AI=靛蓝、能源=琥珀） |
| `{{INDUSTRY}}` `{{EN}}` | 行业中文名 / 英文名 |
| `{{TAGS}}` | 3-4 个该行业核心环节/概念标签，每个 `<span class="tag">…</span>` |
| `{{THESIS}}` | 参考系立框句：一句话点出「摊成地图才看得见瓶颈与价值捕获，且二者常错位」，关键词 `<span class="hl">` 染 accent |
| `{{MAP_SVG}}` | 地图整段 `<svg>…</svg>`（cartographic、节点 placed、🔴🟡 双 markers、错位连线、图例；复用 visual.md 组件） |
| `{{MAP_CAPTION}}` | 地图下一行小注：点破这张图的错位/权力结构（一句话） |
| `{{BASE_RATES}}` | 三个 base rate 的 HTML：每个 `<div class="brate"><div class="blabel">指标名</div><div class="bval">数值</div><div class="bmean">这意味着…</div></div>` |
| `{{Q1}}` `{{Q2}}` `{{Q3}}` | 三个大问题，每个一句话锋利可争论 |

```bash
  /tmp/ljg_map_{name}.html ~/Downloads/{name}-产业地图.png 1080 1440 fullpage
```

## 交付

1. Read 输出 PNG 亲眼验图，**放大看地图板**：节点 placed 且位置有几何理由 ✓、价值流向清晰 ✓、🔴 瓶颈与 🟡 价值捕获两类 marker 视觉可辨 ✓、错位则有连线点破 ✓、图例清晰 ✓、三个大问题在卡上 ✓、accent 协调只点缀 ✓、右侧无空白、底部无留白 ✓。
2. 报告文件路径 + 一句话点出这张地图最锋利的发现（通常是瓶颈与价值捕获的错位）。

## 红线（每条必过）

1. **是地图不是流程图** — 节点有位置关系（上下游/远近/对峙），说得出几何理由；只有方框和连线 = 没画出参考系，回去重做。
2. **两类节点必标且可辨** — 🔴 瓶颈、🟡 价值捕获，两套独立图层；视觉上一眼分得开。
3. **错位必点破** — 瓶颈与价值捕获不在同一节点时，连线 + caption 说破「价值在这创造、钱在那捕获」。
4. **正好三个大问题** — 不多不少，未决张力、指向未来、互不重叠。
5. **deep research 不靠印象** — 价值链结构、利润分布、卡点必须联网查证，不凭印象编；扇出纪律见 research.md。编出来的节点过不了「这环到底卡不卡 / 赚不赚」的复核。
6. **头像是「你在这里」定位针，条件性入图** — 判据：看图人在这张地图上有没有明确位置？有（继刚/用户是这行玩家，或问『我在哪』）→ 在那个节点嵌真墨像（`assets/ljg-portrait.png`，抠底非手绘）+「你在这」+ 指针；无（陌生行业纯鸟瞰）→ 不放。默认 off，绝不当装饰滥放。
7. **base rate 必真查不编** — 三指标的基线值 deep research 查证，区间可、编不可；每项带「这意味着」点破个案幻觉，不留枯燥数字墙。
8. **真渲染真验图** — 必 capture.js 渲染、必 Read PNG 亲眼看；禁「已生成」幻觉（填进正文 ≠ 文件已建）。
9. **反翻译腔 + 禁切痕风** — 卡上 prose 与 caption 默念过关；禁「这一刀」「锋利」（当元描述）「钉死」「砸实」这类自我修辞。
10. **house style 同源** — 浅色玻璃卡身 + cartographic 地图板，与 ljg-library 一家人，不另起视觉。

## Gotchas（务必避开）

- **退化成流程图**：最常见的失手。判据——能不能说出「谁在谁上游、谁卡谁、谁离钱近」。说不出 = 没建参考系，是装饰。
- **瓶颈 = 价值捕获 的默认假设**：默认它们重合是错的。先分别独立找，再看落在哪——错位才是洞见。
- **节点画成公司**：节点是**功能位**（晶圆代工、模型训练、分发），公司只是当下占位者，标在节点旁。换了占位者，位置还在。
- **/tmp 文件名竞争**：并行铸卡共享 `/tmp/ljg_map.html` 会串；用唯一名（带行业 slug）。
- **地图板是手写感 SVG**：线条/网格套 `filter="url(#rough)"` 出 cartographic 手绘抖动；**文字/批注/marker 标签绝不套滤镜**（糊）。组件与骨架见 `references/visual.md`。
- **accent 写死 vs 语义色**：accent 按行业温度选、注入；🔴瓶颈红 / 🟡价值捕获金是**语义固定色**，不随行业变。两套别串。
- **研究扇出挂死**：并发研究 agent 必令同步返回、批量错峰、落盘前 ID 抽样复核（feedback_research_agent_fanout_synchronous）。
